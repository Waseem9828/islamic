# **App Name**: Islamic Random Selector

## Core Features:

- Home Page: Displays the main navigation buttons: 'Bismillah - Start Draw', 'Prayer Times', 'Virtues of Sadaqah', and 'About App'.
- Draw Page - Dua: Presents a prayer ('Allahumma Khir Li Wakhtar Li') before proceeding to the draw settings.
- Draw Page - Settings: Allows users to configure the draw, specifying the range of numbers (1-99 or 1-100) and the number of selections (4 or 5).
- Live Draw Animation: Shows an animation of numbers mixing, with numbers stopping every 2 seconds, accompanied by 'Alhamdulillah' after each number is revealed.
- Result Display: Presents the final selected numbers with the verse 'Fa InnAllaha Huwal Ghaniyyul Hameed', and options to share the result, redraw, or return to the home page.
- Prayer Times Page: Displays the prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha) for the current day.
- Virtues of Sadaqah: Shares the virtues of Sadaqah (charity) based on Islamic teachings.

## Style Guidelines:

- Primary color: Islamic Green (#006400) to reflect Islamic themes and create a peaceful environment.
- Secondary color: White (#FFFFFF) for a clean and pure aesthetic.
- Accent color: Golden (#FFD700) to highlight important elements and provide a touch of elegance.
- Arabic font: 'Amiri' (serif) for the Arabic text.
- Urdu font: 'Jameel Noori Nastaleeq' (serif) for Urdu text.
- English font: 'Roboto' (sans-serif) for English text. Note: currently only Google Fonts are supported.
- Use icons such as a mosque, tasbih (prayer beads), and the Quran to enhance the app's Islamic theme.
- Maintain a clean and organized layout with ample spacing and clear distinctions between sections.
- Use subtle animations during the draw process to engage users and enhance the overall experience.
